# Editor y Preview Markdown.

![editormarkdown](./assets/editormarkdown.png)

Utilizamos **React** y **Webpack**

## Librerias usadas 

- Code mirror
- react-markdown
- github-markdown.css

## Paso a Paso

- Hacemos un _npm install_

```bash
npm install
```

- Para iniciar el proyecto en local:

```bash
npm start
```
